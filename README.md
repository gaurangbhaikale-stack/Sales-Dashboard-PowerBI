# 📊 Product Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Product Sales Dashboard** developed using **Microsoft Power BI**. It provides comprehensive insights into sales performance, customer behavior, product trends, regional performance, discounts, promotions, and yearly sales.

The dashboard enables businesses to monitor important Key Performance Indicators (KPIs) and make data-driven decisions.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Monitor business KPIs.
- Compare sales across regions and stores.
- Evaluate discount and promotion effectiveness.
- Understand customer purchasing behavior.
- Support business decision-making through visualization.

---

## 🛠️ Tools Used

- Microsoft Power BI
- Microsoft Excel
- Python Pandas

---

## 📊 Dashboard KPIs

- Average Shipping Cost
- Returned Products
- Discount Based Products
- Total Orders
- Total Earnings
- Total Sold Products

---

## 📈 Dashboard Features

- Sales by Region
- Orders by Store
- Product Sales Analysis
- Payment Method Analysis
- Discount Comparison
- Customer Type Analysis
- Promotion Analysis
- Product Savings Analysis
- Annual Sales Trends
- Region Manager Performance

---

## 🔍 Key Insights

- East region generated the highest sales.
- Store D received the highest number of orders.
- Discount strategies influenced product sales.
- Online and Cash were among the most preferred payment methods.
- Promotional offers significantly impacted sales.
- Sales trends can be compared across multiple years.
